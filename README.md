# Task Manager API (Spring Boot)

A RESTful API built with Java and Spring Boot to manage tasks.

## 🛠 Tech Stack
* **Java 17+**
* **Spring Boot 3.x** (Web, Data JPA, Validation)
* **PostgreSQL** (Database)
* **Maven** (Build Tool)

## 🚀 Getting Started
1. **Database Setup**: Create a database named `task_db` in PostgreSQL.
2. **Configuration**: Update `src/main/resources/application.properties` with your PostgreSQL username and password.
3. **Run**:
   ```bash
   mvn spring-boot:run
